## Running locally

Clone the project

```bash
  git clone https://github.com/GloomyDarkness/WhatsBot
```

Enter the project directory

```bash
  cd WhatsBot
```

Install the dependencies

```bash
  npm install
```

Start the server

```bash
  node .
```
